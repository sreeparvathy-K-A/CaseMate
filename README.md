**CaseMate - Legal Case Management System**
CaseMate is a web-based application designed to streamline legal case management. It connects clients with advocates, helps manage case details, facilitates appointments, and supports online payments, making legal services more accessible and organized.

Features

 **Admin Module**
  - Manage IPC sections, categories, and courts
  - Manage advocates and clients
  - View and monitor case activities

 **Client Module**
  - Register and login
  - View advocate profiles
  - Chat with advocates
  - Submit new case details
  - Make online payments
  - Provide feedback and ratings

 **Advocate Module**
  - Register and login
  - Add case history
  - Manage appointments
  - Chat with clients
  - Handle online payments



## Technologies Used

- **Backend:** Python  
- **Frontend:** React / HTML / CSS / JavaScript /Bootstrap
- **Database:** MySQL  
- **Version Control:** Git & GitHub
